스레드
    - 용어
        1. 프로그램 : 컴퓨터에게 주어지는 처리 방법과 순서를 기술한 일련의 명령문(코드)의 집합체

        2. 프로세스 : 실행 중인 프로그램

        3. 멀티 태스킹 : 두 가지 이상의 작업을 동시 처리 기술

        4. 멀티 프로세스 : 운영 체제가 동시에 여러개의 프로세스 생성해서 처리 기술
            - 멀티 프로세스가 프로그램 단위의 멀티 태스킹
            - 멀티 프로세스는 서로 독립적이다.(서로에게 영향이 없다.)

        5. 스레드 : 흐름 뜻
            - 프로세스 내에서 실행되는 흐름의 단위 , 코드를 읽어/실행 하는 흐름
            - 싱글 스레드 : 프로세스 내 흐름이 1개
            - 멀티 스레드 : 프로세스 내 흐름이 2개 이상

        6. 멀티 스레드 : 프로그램 내부에서의 멀티 태스킹
        - 워드 프로그램(프로세스) , 파워 포인트 프로그램(프로세스) -> 멀티 프로세스
            - 워드에서 오류가 발생해서 프로그램 종료되면 파워포인트도 종료? x
            - 동시에 사용 하더라도 워드에서  오류가 발생하더라도 파워포인트는 정상
        - 카카오톡(프로세스). 유튜브(프로세ㅐ스) --> 멀티 프로세스
            - 카카오톡 안에서 채팅하면서 사진전송도 동시에 가능
        - 하나의 프로세스 안에서 동시 작업 처리 --> 멀티 스레드
            - 파일 전송시 오류가 발생하면 카카오톡 프로세스가 종료되므로 채팅이 같이 종료
            - 멀티 스레드는 서로 종속관계이다 (서로에게 영향이있다)

    [어려운점]

    앱/웹 개발 시 서버프로그램 내 내장지원을 하기 때문에 신입개발자가 직접 구현하는 경험이 부족할 수 있다.
    실무에서는 구현 보다는 이론이 중요

    싱글 스레드 : 한 개만 존재하는 스레드
        - 자바 코드를 실행하기 위해 내장 싱글스레드를 제공 ->main 함수
    멀티 스레드 : 2개 이상 존재하는 스레드
        - main 함수 스레드 외 추가적인 스레드 생성

    [구현 방법]
        Thread 클래스
            -Runnable 인터페이스를 implements(구현) 클래스
        - Runnable 인터페이스
            추상메소드 : 구현되지 않은 메소드 , 선언부 있는 메소드 , { 구현부 }가 없는 메소드
                run(){} : 스레드(코드를 읽는 흐름) 가 읽을 코드를 정의하는 곳.


    - 멀티스레드 생성 방법
        -1. run 메소드를 재정의한다
        -2. start() 실행한다
    [1]
        Thread thread = new Thread( new Runnable() {
             @Override
             public void run() {
        }) ;
        thread.start();
    [2]
        Thread thread = new Thread(){
            @Override
            public void run() {
                })
        }thread.start();
        }
    [3-1]
        구현클래스 만들기 implements
        class 클래스명 implements Thread{
            @Override
            public void run() {
        }
    [3-2]
        구현클래스 만들기
        class 클래스명 extends Thread{

        }
